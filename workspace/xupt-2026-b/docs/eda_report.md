## 关键结论

- 附件1：两路 Δt 标准差极小（<1e-13）→ 采样稳定无噪声；起始时间差 247.8s
- 附件2：两路二阶差分估计噪声 σ ≈ 0.80-0.86 m；公共时段 639s 很充分
- 附件3：噪声 σ 方式1≈4.18 m，方式2≈2.89 m（方式1 噪声更大，或含更多异常点）

### ⚠ 附件3 时段完全不重叠
- 方式1 t ∈ [469.050, 814.050]
- 方式2 t ∈ [77.130, 401.130]
- 两路公共时段 = ∅；原始时间戳差 Δτ_naive = -391.92s

→ **这不是 "无公共时段无法融合"，而是 "Δτ 范围在 [+300, +450] 附近"**（方式2 的时间 + Δτ ≈ 方式1 的时间）。问题 3 的 Δτ 搜索域应设为 [+300, +500]，估出后方式2 的时间 + Δτ̂ 才与方式1 匹配。

### 问题 4 输入注意
问题 3 融合输出的 10Hz 轨迹时间跨度 ≈ min(345, 324) = 324s，对应问题 4 决策网格：
- 2Hz: 648 时刻 → 基础变量 N_s·N_t + N_p·M·N_t ≈ 18·648 + 18·M·648
- Δθ=45°(M=8): ≈ 105k，可行窗剪枝后预计 <20k
- CP-SAT 首选；CBC 兜底；贪心基线必出。
