\abstracttitle

多源异构传感器融合定位是自主移动机器人实现高精度导航的关键技术，面临采样频率差异、时间偏移及观测噪声等挑战。本文针对两路异频定位数据（4\,Hz 与 5\,Hz）的时空对齐、噪声融合与任务规划问题，建立了四层递进的数学模型体系。

\textbf{针对问题一，}两路数据无噪声，仅存在开机时间差。对两路分别构建三次样条插值（$C^2$ 连续），以残差平方和为目标建立优化模型，采用粗搜（$\pm 50$\,s，步长 0.1\,s）加黄金分割精搜（精度 $10^{-6}$\,s）求解，复杂度 $O(N\log N)$。最优偏差 \textbf{$\Delta\tau = -198.43$\,s}，对齐后 \textbf{RMSE $= 8.39\times10^{-11}$\,m}，输出 \textbf{10\,Hz} 轨迹 \textbf{7004} 点。

\textbf{针对问题二，}两路含噪声（$\sigma \approx 0.8$\,m）与固定偏差。建立联合最小二乘模型同时估计 $\Delta\tau$ 与 $(\Delta x, \Delta y)$，避免分步误差传播。Nelder-Mead 求解得 \textbf{$\Delta\tau = -48.44$\,s}，\textbf{$\Delta x = -3.44$\,m}，\textbf{$\Delta y = 1.81$\,m}，残差 \textbf{RMSE $= 1.56$\,m}。设计常加速度卡尔曼滤波器异步融合两路观测，输出 \textbf{8513} 点融合轨迹。

\textbf{针对问题三，}对真实数据构建 Wald 检验与 AIC 双判据框架判定偏差存在性。Wald 统计量 \textbf{$W = 2.07$}，\textbf{$p = 0.356 > 0.05$}；\textbf{$\Delta$AIC $= -2.81 < 2$}，双判据一致判定\textbf{无显著系统偏差}。按零偏模型融合输出 \textbf{3691} 点轨迹。

\textbf{针对问题四，}对融合轨迹施加 151 点 SG 平滑后差分得运动学参数。射击约束：$d \in [5, 30]$\,m，$v \le 2$\,m/s，$a \le 1.5$\,m/s$^2$，准备 1.5\,s；拍照约束：$d \in [10, 40]$\,m，$v \le 1.5$\,m/s，$a \le 1.5$\,m/s$^2$，准备 0.5\,s，角差 $\ge 60°$。贪心调度 \textbf{1.2\,s} 完成：\textbf{射击 8 次，拍照 16 次，共 24 个任务}，逐任务验证全部满足约束。

\textbf{创新点：}① 联合参数估计将时间与空间偏差纳入同一优化目标，避免分步误差传播；② Wald + AIC 双判据从统计显著性与模型选择两维度判定偏差；③ SG 平滑与贪心调度联合框架，灵敏度分析确定最优窗口；④ 各环节经独立数值验证，17 项检查全部通过。
